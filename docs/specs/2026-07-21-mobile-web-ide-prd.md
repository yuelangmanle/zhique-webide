## 一、产品概述

### 1.1 产品定位

Mobile Web IDE 是一款运行在 Android 移动端的 Web 开发工具，允许用户在手机上编写、运行和调试 HTML/CSS/JavaScript 代码，并将其打包成可安装的 Android APK 文件。

### 1.2 核心价值

- **随时随地开发**：在手机上完成 Web 应用的全流程开发
- **一键打包**：将 Web 代码直接打包成 APK，无需电脑
- **原生能力调用**：支持调用 Android 原生权限（相机、位置、蓝牙等）
- **AI 辅助**：集成大语言模型，辅助代码生成和调试

### 1.3 目标用户

- 前端开发者（需要在移动端快速开发和测试）
- 独立开发者（资源有限，需要低成本开发工具）
- 技术爱好者（学习 Web 开发的移动端工具）

## 四、技术方案概述

### 4.1 架构模式

采用模块化组件化架构，每个功能模块独立开发、测试和部署。

### 4.2 核心技术栈

| 模块 | 技术 | 版本 |
|------|------|------|
| 主框架 | Capacitor | 6.x |
| 代码编辑器 | CodeMirror 6 | latest |
| 实时预览 | Android WebView | 系统自带 |
| AI 接口 | 多模型 API | 配置化 |
| 打包引擎 | Gradle | 8.x |

### 4.3 关键技术挑战

1. **移动端 APK 构建**：在 Android 上运行 Gradle 和 Android SDK 组件
2. **权限管理**：同时支持运行时动态授权和打包前预配置
3. **代码编辑器性能**：在移动端实现 IDE 级别的编辑体验
