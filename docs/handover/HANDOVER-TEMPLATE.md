## 四、开发流程

### 4.1 命令说明

| 命令 | 描述 |
|------|------|
| `npm run dev` | 启动开发服务器 |
| `npm run build` | 构建生产版本 |
| `npm run lint` | 代码质量检查 |
| `npm run test` | 运行测试 |
| `npm run cap:sync` | 同步到 Android |
| `npm run cap:build` | 构建 APK |

### 4.2 分支策略

- `main`: 主分支，稳定版本
- `develop`: 开发分支，功能集成
- `feature/*`: 功能分支
- `bugfix/*`: 修复分支

## 五、模块说明

### 5.1 代码编辑器模块

**职责**: 代码编辑功能
**位置**: `src/editor/`
**技术**: CodeMirror 6

### 5.2 实时预览模块

**职责**: 实时预览功能
**位置**: `src/preview/`
**技术**: WebView + IFrame

### 5.3 AI 助手模块

**职责**: AI 代码生成和调试
**位置**: `src/ai-assistant/`
**技术**: Axios + 多模型 API

### 5.4 APK 打包模块

**职责**: APK 构建功能
**位置**: `src/apk-builder/`
**技术**: Capacitor Plugin + Gradle

### 5.5 权限管理模块

**位置**: `src/permission-manager/`
